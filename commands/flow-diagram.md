# Slash Command: /flow:diagram

## Purpose
Convert n8n workflow connection structures into clean, compile-ready Mermaid Flowchart text scripts.

## Execution Guide
```bash
flowforge diagram <filepath> --out <output-filepath>
```

## Guidance
*   Render output diagrams as markdown previews using ` ```mermaid ` tags.
*   Clearly map node types to styling boxes (Triggers, Logic, Code, Actions).
