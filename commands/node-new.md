# Slash Command: /node:new

## Purpose
Scaffold custom n8n node directories using official node starter structures.

## Execution Guide
```bash
flowforge node-new <package-name> --auth <auth-type> --resource <resource-name> --operation <operations-list>
```

## Guidance
*   Ensure the generated node points out TODO comments for custom integration parameters.
*   Prompt the user to review the files inside `custom-nodes/` directory.
