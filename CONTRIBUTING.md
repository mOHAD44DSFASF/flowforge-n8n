# Contributing to FlowForge n8n

We welcome contributions to FlowForge n8n!

## Guidelines
1.  **Fork the repo:** Fork to your account and branch off of `main`.
2.  **Run tests locally:** Always execute `pnpm test` and `pnpm lint` before submitting pull requests.
3.  **No secrets:** Ensure all testing configurations and credentials are sanitized.
4.  **Follow specifications:** Align node layouts with schemas defined in `SPEC.md`.
